# Locus Review Demo

This folder contains a tiny public demo dataset for the local Locus Review workbench.

The molecule images are synthetic RDKit drawings generated for this repository. They are not competition images and are safe to keep in the public repository.

Run the demo with:

```bash
python tools/launch_review_editor.py --demo
```

中文说明：这个目录是 Locus Review 工作台的公开演示数据。里面的分子图片是为本仓库生成的合成 RDKit 图片，不是比赛图片。
